# colores-py

# Tkinter & Python Colors Chart

## ¿Qué es esto?
Esto es tan solo una herramienta bastante útil para identificar los nombres de los colores a la hora de realizar un proyecto.

## ¿Cómo puedo ejecutarlo?
Para ejecutarlo debes disponer de *python* y *tkinter* en tu pc, una vez que dispongas de esto debes ejecutar el comando *python colors.py*
